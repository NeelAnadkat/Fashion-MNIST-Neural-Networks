# Fashion-MNIST Classification: ANN vs. CNN Comparative Analysis

## 📌 Project Overview
This project evaluates and compares two deep learning architectures—**Artificial Neural Networks (ANN)** and **Convolutional Neural Networks (CNN)**—using the Fashion-MNIST dataset. This was completed as part of my **MSc in Artificial Intelligence** at **De Montfort University**.

## 📊 Key Results
* **CNN Accuracy:** 93.07% (Test Set)
* **ANN Accuracy:** 88.98% (Test Set)
* **Observation:** The CNN outperformed the ANN by 4.15% due to its superior ability to learn hierarchical spatial features in image data.

## 🛠️ Technical Implementation
- **Frameworks:** TensorFlow, Keras, NumPy
- **Visualization:** Matplotlib, Seaborn
- **Techniques Used:** - 17-layer CNN architecture with 3-block feature extraction
  - Batch Normalization & Data Augmentation
  - Regularization: Dropout, Early Stopping, and Learning Rate Reduction
